# Automate
Automate , a tool for handling git ops , jira , bitbucket and day to day workflows 

Installation: 

npm i -g automate-cli@latest

Usage: 
Open terminal / cmd 
Type: automate -> hit enter
Then follow the prompts

Publishing 
npm login 
npm whoami ( optional )
npm publish 
npm version patch
npm version minor
npm version major

Test
npm link automate
automate
npm unlink automate
